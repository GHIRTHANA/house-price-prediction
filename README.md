This project is a House Price Prediction System built using Machine Learning in Python. The model predicts house prices based on housing-related features such as area, number of bedrooms, bathrooms, floors, location, condition, year built, garage availability, and other property details. The project includes data preprocessing, feature encoding, model training, evaluation, and house price prediction.

Features:
1.House price prediction using Machine Learning
2.Data preprocessing and cleaning
3.Feature encoding and transformation
4.Model training and evaluation
5.Prediction based on property details

Technologies Used:Python,Pandas,Scikit-learn,
